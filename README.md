# QookingFever
Pipeline for converting FastQ files into count matrix

## User Guide
1. run:
```
cookiecutter git@github.com:Hideyuki-Okano-Lab/QookingFever.git
```
2. work in progress... 

---
## For developpers
0. prerequisite: `poetry`
1. clone this repository
2. run:
```
poetry install
poetry run pre-commit install
```
